# Angles
Code editor based on Ace ([website](http://ace.c9.io))

## Main features
 - Multi-tab editing
 - Syntax highlighing
 - JS, CSS and HTML beautifiers
 - Toolbar show/hide
 - Full adaptation to AtomOS UI
